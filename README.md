This project automatically organizes files in your Downloads folder into categories like Images, Documents, Videos, and Others.

>>Features

Automatically detects new files
Moves files into correct folders
Easy to run and customize
Useful automation project for beginners

>>How to Run
Install the required package:
pip install watchdog


Change this line in organizer.py:
WATCH_FOLDER = r"C:\Users\YourName\Downloads" #your path address

Run the script:
python organizer.py

Paste or download any file into your Downloads folder
